## ls

全称 list，显示一个文件夹下的各种内容

| 参数                | 说明                                |
| ------------------- | ----------------------------------- |
| -a (-A)             | 显示所有文件夹（但不列出 .  和 ..） |
| -h --human-readable |                                     |
| -s                  | 以 block 为单位显示文件大小         |
| -l                  | 显示详细信息                        |
| -R                  | 递归显示所有文件（包括子目录中的）  |
| -t                  | 以文件修改时间排序                  |
| -X                  | 按扩展名排序                        |
